# トランスポート福岡 新着通知（通知専用 拡張）

新規リードを **Windows通知**でお知らせするだけの軽量拡張です。
監視（ズバット／引越し侍のスクレイピング）は**しません**。各スタッフのPCに入れる「子」拡張です。

## 特徴
- **CRMを開かなくても**通知が届く（Chromeが起動していればOK）
- **ログイン不要**・**監視サイトのタブ不要**
- 新着を約12秒間隔でチェックして通知（初回インストール後に来た新着のみ）

## できないこと
- リードの監視・取得はしません（それは親拡張＝`extension/` の役割）
- Chromeを完全に終了していると通知は出ません（Chrome起動中のみ）

## インストール手順（各PC）
1. このフォルダ `extension-notify/` をPCに置く（リポジトリを clone でも可）
2. Chromeで `chrome://extensions` を開く
3. 右上の「**デベロッパーモード**」をオン
4. 「**パッケージ化されていない拡張機能を読み込む**」→ `extension-notify` フォルダを選択
5. Windowsの「設定 → システム → 通知」で **Google Chrome の通知をオン**、集中モード/応答不可をオフ

## 使い方
- インストールするだけで通知が始まります。
- 拡張アイコンのポップアップで **ON/OFF切替** と **テスト通知** ができます。
- 通知をクリックすると CRM（transportfukuoka.vercel.app）を開きます。

## 仕組み
`https://transportfukuoka.vercel.app/api/inbound?recent=5` を定期取得し、
前回以降に来た新着（`savedAt` が新しいもの）だけを通知します。
